What is this?

While there is a growing corpus of formalised theorems including proofs, there is a lack of open conjectures where only the statement has been formalised. This repository collects such statements from diverse sources — Erdős’s problem lists, Wikipedia, MathOverflow, the OEIS, research papers, and more.

The project aims to become a benchmark for automated theorem provers, help clarify conjectures through formalisation, and highlight gaps in Mathlib.
